Points Criteria

  • 1. On every run of a player 1 point will be awarded to participant
  • 2. Player Scoring 50+ runs will be awarded 25 points
  • 3. Player Scoring 100+ runs will be awarded 50 points.
  • 4. On each six 3 points will be awarded
  • 5. On each four 2 points will be awarded
  • 6. On each wicket 20 points will be awarded
  • 7. If a bowler takes 3 or more wickets than 30 points will be awarded
  • 8. If a bowler takes 5 or more wickets than 50 points will be awarded
  • 9. If a bowler takes 7 or more wickets than 100 points will be awarded
  • Bonus Points: 1) If a player is Man of the Match than 100 points will be awarded
  • 2) All of the above points will be x2 if a player is captain
  • Enjoy the game and All The Best
